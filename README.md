# MMSGI Starter Kit

CLI untuk membuat project web Go dengan template admin siap pakai.

## Install

go install github.com/mmsgi-library/mmsgi-starter-kit/cmd/mmsgi@latest


## Generate Project Baru

mmsgi new myadmin
cd myadmin
go run cmd/server/main.go


Server otomatis jalan di:

http://localhost:8080
